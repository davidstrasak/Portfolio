Lessons learned:

  • Lots of debugging and installing operating systems.
  • Learned in general about how the internet and websites work.
  • Installing and using Armbian.

Description:

This was a summer 2023 project that involved setting up a Banana Pi to use PiHole on.

PiHole is an application that monitors internet traffic and blocks trackers, advertisements or anything else you want blocked. After installation it acts as a server and is online 24/7.
